계정 생성 방법

Drinkly Privacy는 다음과 같은 계정 생성 방법을 지원합니다:

  • 사용자 email 및 비밀번호
  • OAuth (Google 계정 연동)

계정 삭제 요청

사용자는 언제든지 자신의 계정과 관련된 모든 데이터의 삭제를 요청할 수 있습니다.

계정 삭제 절차

  1. 앱 내 설정 메뉴에서 '계정 삭제' 옵션 선택
  2. 또는 이메일로 삭제 요청: dhdnjswnd@gmail.com

삭제되는 데이터

  • 계정 정보 (사용자 이름, 이메일 등)
  • 음주 기록 데이터
  • 설정 및 환경 설정

데이터 삭제는 요청 후 30일 이내에 처리됩니다.

부분 데이터 삭제

사용자는 계정을 유지하면서 특정 데이터만 삭제할 수 있습니다:

  • 특정 날짜의 음주 기록 삭제
  • 특정 기간의 데이터 삭제
  • 모든 음주 기록 삭제 (계정 유지)

부분 데이터 삭제 방법

  1. 앱 내 '기록 관리' 메뉴 접속
  2. 삭제하고자 하는 데이터 선택
  3. '삭제' 버튼 클릭

데이터 보관 정책

삭제 요청된 데이터는 다음과 같이 처리됩니다:

  • 즉시 삭제: 사용자 개인정보, 음주 기록
  • 30일간 보관 후 삭제: 백업 데이터
  • 법적 요구사항에 따른 보관: 결제 기록 (관련 법률에 따라 5년간 보관)
